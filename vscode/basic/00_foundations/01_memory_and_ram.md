# メモリ（RAM）の基礎

## RAM とは

- **Random Access Memory** = どの位置にも同じ時間でアクセス可能な高速メモリ
- 実行中のプログラム / データの置き場所

## メモリの階層 (速い → 遅い)

```
┌─────────────┐  数ナノ秒
│  CPU Cache  │
├─────────────┤  数十ナノ秒
│     RAM     │
├─────────────┤  ミリ秒
│  SSD / HDD  │
└─────────────┘
```

## アドレスのイメージ

```
arr = [1, 2, 3, 4, 5]

Index :   0      1      2      3      4
Addr  : 0x100  0x108  0x110  0x118  0x120
Value :   1      2      3      4      5
         ↑
   id(arr[0]) で確認できる (Pythonは内部表現が異なる)
```

→ 実コード例は `examples_array.py` の `demo_memory_address()` 参照
