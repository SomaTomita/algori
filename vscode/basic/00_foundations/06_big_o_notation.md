# Big O 記法

## Big O とは

入力サイズ N が増えたとき、**処理時間がどう伸びるか** を表す尺度。
定数や下位項は無視して、**最も効くもの** だけ残す。

## よく出る計算量

| Big O        | 名前         | 典型例                                    | コード片                             |
| ------------ | ------------ | ----------------------------------------- | ------------------------------------ |
| O(1)         | 定数         | 配列の index, dict / set の lookup        | `arr[5]`, `s in seen`                |
| O(log n)     | 対数         | 二分探索, バランス木                       | `while l <= r: mid = (l+r)//2`       |
| O(n)         | 線形         | 1 重ループ, 配列スキャン                   | `for x in arr: ...`                  |
| O(n log n)   | 線形対数     | 効率的なソート (merge / quick / Timsort)   | `arr.sort()`                         |
| O(n²)        | 二乗         | 二重ループ, 全ペア比較                     | `for i: for j: ...`                  |
| O(2ⁿ)        | 指数         | 素朴な再帰 (素朴フィボナッチなど)         | `fib(n-1) + fib(n-2)`                |

## 増え方の視覚化

```
時間
 ↑
 │                                    O(2^n)  ← すぐ天井
 │                              O(n^2)
 │                        ╱
 │                  ╱          O(n log n)
 │            ╱           ╱
 │      ╱           ╱           O(n)
 │ ╱           ╱           ╱
 │       ╱           ╱             O(log n)
 │─────────────────────────────────  O(1)
 │
 └──────────────────────────────────→ N (入力サイズ)
```

## 入力サイズの目安 (1 秒制約)

```
N の上限         | 必要な計算量
─────────────────|─────────────────────
N <= 10         | O(N!), O(2^N)  まで OK
N <= 20         | O(2^N)  まで OK
N <= 500        | O(N^3)  まで OK
N <= 5,000      | O(N^2)  まで OK
N <= 10^4       | O(N^2)  ぎりぎり (定数次第)
N <= 10^5       | O(N log N)  まで欲しい
N <= 10^6       | O(N) または O(N log N)
N <= 10^9       | O(log N) / O(1) 以外無理
```

## ルール・オブ・サム

- **N >= 10^5 なら O(N²) は捨てる** → sliding window / hashmap / 二分探索を考える
- **見つけた解の Big O を必ず口に出す** (面接でも実務でも)
- **平均 / 最悪** を区別する (hashmap は平均 O(1), 最悪 O(N))

## 空間計算量も忘れずに

時間と同じ記号で **追加で使うメモリ** を表す。

```
配列をコピーして並べ替え     → O(N) extra space
ポインタ 2 つで in-place    → O(1) extra space
再帰の深さ h               → O(h) スタック
```
