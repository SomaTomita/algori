# 集合 (Sets)

## set とは

- **重複なし** のコレクション
- 順序なし
- **`in` 検索が平均 O(1)** で超高速 (内部はハッシュテーブル)

```python
s = set([1, 2, 3, 3, 2])
# s == {1, 2, 3}    # 重複は自動で消える

3 in s              # True   平均 O(1)
s.add(4)            # 平均 O(1)
s.remove(1)         # 平均 O(1)
```

## list vs set

```
┌──────────────┬───────────┬──────────┬───────────┐
│   操作        │   list    │   set    │   勝者     │
├──────────────┼───────────┼──────────┼───────────┤
│ in 検索       │   O(N)    │  O(1)*   │   set     │
│ 重複排除      │   手動    │  自動    │   set     │
│ 順序保持      │    Yes    │   No     │   list    │
│ index アクセス│    O(1)   │   不可   │   list    │
└──────────────┴───────────┴──────────┴───────────┘
* 平均 (ハッシュ衝突最悪は O(N))
```

## 用途リスト

- **uniqueness** : 重複を取り除く / 重複があるか判定
- **existence** : 「これ前に見た?」を高速に判定
- **fast membership** : ループ内で何度も `in` する場面
- **sliding window unique** : 窓の中の重複文字を追跡 (後章)
- 集合演算 : `a | b` (和), `a & b` (積), `a - b` (差)

## 注意: key/value が必要なら dict

set は値の集合のみ。「単語 → 出現回数」など **key と value のペア** が
必要なら **`dict`** を使う (次章 hashmaps で扱う)。

```python
seen = set()                # 値だけ覚える
count = {}                  # 値 + 出現回数を覚える
```
